#4-dev-warn
===========
В этой задаче добавляем функциональность:  
  -  `compile/warning`  
  -  `compile/magic`  

--------------------------------------------------------------------------------

Инструмент `compile/warning.hpp` служит для кросс-платформенного  
подавления предупреждений компилятора  

```cpp
#include <compile/warning.hpp>

dPRAGMA_PUSH
dPRAGMA_DISABLE(dWRN_UNUSED_VARIABLE)

int main()
{
    int val;
}

dPRAGMA_POP
```

--------------------------------------------------------------------------------

Инструмент `compile/magic.hpp` был создан просто ради забавы.  
Позволяет писать код так, что бы он дополнительно отображался в логе сборки.  

Практического значения не имеет, и служит просто для наглядного примера,  
как на языке с++ можно проворачивать подобные трюки  

```cpp
#include <compile/magic.hpp>
#include <iostream>
using namespace std;
 
int main()
{
    cout << "started...\n";
    int Fedya = 40;
    dMAGIC(cout << "Fedya = " << Fedya << endl);
}
```
