Description: features of compile  
Description: особенности компиляции  

--------------------------------------------------------------------------------

Библиотека предназначена для определения свойств компилятора,  
и управления сборкой на этапе препроцессора.  

Содержание:  
  - `support.hpp` возможности компиляторов  
  - `dfunction.hpp` макрос для вывода имени функции  
  - `warning.hpp` подавление предупреждений  
  - `magic.hpp` печатает исходный код в лог сборки  
  - `info.hpp` логгирование компиляции  

Логгирование компиляции:  
  - `dABOUT_CONFIG(name)` информация о конфигурации  
  - `dSHOW_BUILD(title, name)` информация о сборке  
  - `dCOMPILER_MESSAGE(...)`   не отключаемый  
  - `dPRAGMA_MESSAGE(...)`     отключаемый  

--------------------------------------------------------------------------------

`support.hpp` содержит множество макросов,  
при помощи которых можно определить поддерживает ли компилятор те,  
или иные возможности.  

--------------------------------------------------------------------------------

`dfunction.hpp` содержит макрос `dFUNCTION`  
макрос раскрывается в имя фукнкции, изнутри которой он был запущен.  
служит для изготовления различных отладочных инструментов.  

--------------------------------------------------------------------------------

`warning.hpp` предназначен для кросс-платформенного  
подавления предупреждений компиляторов  

--------------------------------------------------------------------------------

`magic.hpp` позволяет писать код, текст которого  
будет дополнительно отображен в логе сборки  

Эта забавная фича скорее всего не будет использоваться на бою,  
однако была добавлена в библиотеку в качестве наглядного примера,  
как на языке с++ можно проворачивать подобные трюки  

--------------------------------------------------------------------------------

`info.hpp` содержит макросы для логгирование компиляции:  
  - вывод произвольного текста в лог сборки  
  - вывод кратконй информации в лог сборки  

Пример вывода информации в лог сборки:  
```
1>------------------------------------[sample]
1>__cplusplus ....... 201703L
1>c++17 ............. enabled
1>msvc2019 .......... 16.11.19
1>_MSC_FULL_VER ..... 192930146
1>_MSVC_LANG ........ 201703L
1>_MSC_EXTENSIONS ... enabled
1>  ---
1>[sample] disabled -> pch
1>[sample] STABLE-DEBUG
1>[sample] MTd (debug-static runtime c++)
1>[sample] 0.0.5, x86-debug-MTd, stable
1>  ---
```
